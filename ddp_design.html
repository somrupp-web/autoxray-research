Launch Sequence (launch_ddp.py)
1

Prepare all nodes in parallel

SSH into all 4 nodes simultaneously. Check & install open-clip-torch and transformers. Upload train_ddp.py, prepare.py. Pre-download BiomedCLIP weights from HuggingFace Hub so they're cached before training starts.

2

Launch training on all nodes simultaneously

SSH exec on all 4 nodes at once via ThreadPoolExecutor. Each node runs RANK=N WORLD_SIZE=4 MASTER_ADDR=10.137.203.228 MASTER_PORT=29500 python train_ddp.py. NCCL routes gradient traffic over NCCL_SOCKET_IFNAME=enp1s0f0np0.

3

Rendezvous via env:// (TCPStore)

dist.init_process_group(backend='nccl', init_method='env://'). Node-0 (rank 0) starts the TCPStore server on port 29500. Ranks 1–3 connect as clients. All 4 confirm world_size=4 before training begins.

4

Synchronized training budget

dist.barrier() ensures all ranks start their 600s timer simultaneously. Each rank runs its shard of the data through BiomedCLIP + classifier head. Gradients are AllReduced after each backward pass. All ranks exit the loop at the same epoch boundary.

5

Evaluation & results (rank 0 only)

After dist.barrier(), rank 0 runs evaluate_auc() on the validation set, saves trained_model.pth, and prints final metrics. launch_ddp.py streams all rank-0 output back to the laptop and summarizes val_auc.
